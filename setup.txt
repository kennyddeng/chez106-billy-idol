1. install Twilio library (pip install twilio)
2. create Twilio account (https://www.twilio.com/)
3. create Twilio virtual number 
4. configure keys.py with Twilio account id, token, and virtual number
5. configure keys.py with target phone number
6. configure config.py with chosen sample rate and Artist/Album/Song
7. run main.py
8. enjoy (or not)
